# Pipelines

Canonical runnable pipelines:

1. `protenix-a5b1`
- A5B1 heterodimer-focused inputs and notebook workflow.

2. `protenix-avb3-template`
- AVB3 template registration and Protenix template-enabled inference workflow.

3. `boltz`
- Boltz sweep pipeline for extended-conformation search.

4. `afcluster`
- AFCluster-style MSA clustering + Boltz backend sampling.
