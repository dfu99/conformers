# AGENTS.md

## Intent-Specific Memory (tasks/)
For requests in this repository, consult intent-specific memory under `tasks/` before making changes.

### Which file to consult
1. New features, refactors, or reorganizations:
`tasks/planning.md`
2. Running existing pipelines, debugging failures, or interpreting errors:
`tasks/lessons.md`
3. Experimental design updates and method comparisons:
First `tasks/planning.md`, then append outcomes to `tasks/lessons.md`.

### Required maintenance
- After any major scope change, update `tasks/planning.md`.
- After any resolved failure or repeated failure pattern, update `tasks/lessons.md`.
- Keep entries concrete: command context, symptom, likely cause, and action.
